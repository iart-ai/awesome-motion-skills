# Awesome Motion Skills [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of **AI-agent skills for motion graphics, animation, and video** — kinetic typography, data-viz, explainers, social video, web motion, WebGL, and math animation.

A [skill](https://github.com/vercel-labs/skills) is a portable workflow file (`SKILL.md` + references) that teaches an AI coding agent — Claude Code, Cursor, Codex, and 40+ more — *how* to produce a specific kind of motion, with a deliver-and-verify loop so the agent checks its own frames. Most install in one line:

```bash
npx skills add <owner>/<repo>     # or:  /plugin marketplace add <owner>/<repo>
```

This list is an **open aggregator** — motion/animation/video skills from anyone are welcome. See [Contributing](#contributing).
